HW6 assignment:

This is a homework 6 assignment for GUI class. I used JQuery validator plug in to validate the entries for multiplication
table. The main idea is to check for input errors as the user make the error and display the appropriate error message
where the error is made. 
I was able learn about JQuery validator plug in from www.w3schools.com/  , jqueryvalidation.org/validate  , and stackoverflow.com to be able to complete the assignment
